DiscreteWavelets Toolbox

MSE

Mean squared error

Syntax

	e=MSE(A,B)

Description

e=MSE(A,B) MSE takes two matrices A and B as input and returns the mean squared error between the two matrices.

Remarks

The matrices A and B must be of the same dimension.

The DiscreteWavelets Toolbox uses MSE to find the peak signal-to-noise ratio of two matrices.

Example

Compute the mean squared error of two matrices. The commands 

A=magic(3) %Create a 3 x 3 matrix
B=A+rand(3,3) %Add some small values to A
e=MSE(A,B)
produce 

A =

     8     1     6
     3     5     7
     4     9     2


B =

    8.4652    1.2183    6.8214
    3.9191    5.2704    7.9299
    4.4600    9.0094    2.4971


e =

    0.3534
