Entropy

Entropy of a matrix or vector

Syntax

	y = Entropy(v)

Description

y=Entropy(v) takes as input a matrix or vector v and returns the entropy of the input.

Remarks

The entropy is computed by forming a vector of relative frequencies f of the distinct values of v and then dotting f with log2(1/f). See Section 3.3 of the book for more details.

Example

The entropy of a constant vector is 0 

v=ones(64); %Constant vector
e=Entropy(v) %Compute the entropy


e =

     0

The entropy of an n-vector whose elements are distinct is log2(n) 

v=1:64; %Construct a vector of distinct elements
e=Entropy(v) %Compute the entropy


e =

     6
Other vectors of length 64 have entropies in the interval [0,6] 
v=round(10*rand(1,64)); %Construct a vector
e=Entropy(v) %Find the entropy


e =

    3.3160
