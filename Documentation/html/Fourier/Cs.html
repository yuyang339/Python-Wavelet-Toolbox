Cs

Fourier coefficients for cos(t)

Syntax

	v = Cs()

Description

v=Cs() returns the a vector whose components are the Fourier coefficients for the %Fourier series of

cos(t) = .5*eix + .5*e-ix

This form is particularly useful for expanding expressions to create Fourier series.

Example

We can use Cs to find the Fourier coefficients of cos^2(t). 

v=Cs()' %Get the Fourier coefficients for cos(t)
y=conv(v,v) %Use conv to multiply cos(t)*cos(t)


v =

    0.5000         0    0.5000

y =

    0.2500         0    0.5000         0    0.2500
