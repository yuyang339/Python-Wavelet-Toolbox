DiscreteWavelets Toolbox    

What's New?

This toolbox, DiscreteWavelets Version 1.0, represents the first port of the Mathematica DiscreteWavelets package to Matlab. So in essence, everything here is new! For those of you who have used the Mathematica package, the following list gives changes/additions made in/to Version 5.2 to obtain the most current Version 6.0.

Revision of ImageNames Function

The ImageNames function no longer allows the user the option of displaying thumbnails of the images available for use with the DiscreteWavelets. The new functions ShowThumbnails can be used to display thumbnails.

Addition of ShowThumbnails Function

The function ShowThumbnails allows the user to display all grayscale image and/or color images that are available with the DiscreteWavelets Toolbox.

Revision of HuffmanTree Function

The HuffmanTree function has been overhauled and is now much more efficient at displaying Huffman codes. The user now has control of size, color, and fonts for nodes.

Revision of WaveletVectorPlay Function

The WaveletVectorPlay function has been rewritten so that the output is organized in a more logical manner.

Addition of New Functions for De-Noising Applications

The following functions are now available for use in de-noising applications:

NoiseEstimate

This function takes an input vector or matrix and an orthogonal wavelet filter and returns an estimate of the noise level contained in the input.

SureShrink

This function takes an input vector or matrix, an orthogonal wavelet filter, and the number of iterations of the discrete wavelet transformation to be performed, and uses the SureShrink method (see Section 9.3) to return a de-noised version of the input.

TestSparseness

This function can be used to test an input vector or matrix for sparseness (see Section 9.3 of the text) and returns the proper tolerance value.

UniversalThreshold

This function computes the Universal Threshold tolerance for use in the VisuShrink method (see Section 9.2).

WaveletShrinkage

This function takes an input vector or matrix, an orthogonal wavelet filter, a tolerance, and the number of iterations of the discrete wavelet transformation to be performed, and returns a de-noised version of the input.

Addition of Make3D and Split3D Functions (Matlab)

To facilitate work with color images, the functions Make3D and Split3D have been added to the toolbox. Make3D takes three r x c matrices, presumably R,G,B or Y,Cb,Cr channels, and creates a 3-D array that can be used for plotting. The function Split3D returns three r x c matrices from a 3-D array input.
