Installing DiscreteWavelets

The following steps will allow you to install DiscreteWavelets:

  1. Download DiscreteWaveletsPackage60.zip from the website http://www.stthomas.edu/wavelets.
  2. The zip file contains a single folder called DiscreteWavelets. Place this folder anywhere you like on your computer, but do not rename it. Make note of the folder location.
  3. Open Matlab R2007b and under File, choose Set Path …
  4. Click the button Add with SubFolders … and enter the location of the DiscreteWavelets folder.
  5. Click on Save and Close.
  6. In the Matlab command window, issue the command rehash and then startup. This will load the global variables used by the package.
  7. You are all set - you should be able to run demos or use any function in the package.

Adding New Image Files to the Package

To add new images to the package:

  1. In the Matlab command window, type startup and make note of the value of DWIMAGESPATH
  2. Go to this folder and open the folder GrayScale (Color) if you wish to add grayscale (color) images.
  3. Place a copy of the image in the appropriate directory. You should also add a thumbnail of the image as well. The naming convention for the thumbnail is image_small.png where the added image is named image.png. You may use any format supported by the Matlab function imread.
  4. Open the text file Images.txt with a text editor. Add a line of the form image.png;r(r_small);c(c_small);i(i_small) for each new image you have added. Here the image has dimensions r x c and the thumbnail has dimensions r_small x c_small. A total of i iterations of the discrete wavelet transformation can be performed on image.png and i_small iterations can be performed on image_small.png.
  5. Save and close the file. In the Matlab command window, issue the command ImageList(). The added image files(s) should show up in the output table.

Adding New Sound Files to the Package

To add new sounds to the package:

  1. In the Matlab command window, type startup and make note of the value of DWSOUNDSSPATH
  2. Go to this folder and place a copy of the sound file(s) in it. You may use either the .wav or the .au sound file format.
  3. Open the text file Sounds.txt with a text editor. Add a line of the form audio.wav;TYPE;len;channel;srate;size for each new sound file you have added. Here, the first element of the list is the file name, TYPE is the type of audio file ("WAVE", for example), len is the length of the audio clip in seconds, channel is the number of channels, srate is the sampling rate, and size is the number of samples in the file.
  4. Save and close the file. In the Matlab command window, issue the command SoundList(). The added sound file(s) should show up in the output table.

Adding New Data Files to the Package

To add new data files to the package:

  1. In the Matlab command window, type startup and make note of the value of DWDATAPATH
  2. Go to this folder and place a copy of the data file(s) in it.
  3. Open the text file Data.txt with a text editor. Add a line of the form Add a line of the form data.txt;size;comment; for each new data file you have added. Here, the first element of the list is the file name, size is the the number of samples in the file, and comment is a comment about the data in the file.
  4. Save and close the file. In the Matlab command window, issue the command DataList(). The added data file(s) should show up in the output table.
