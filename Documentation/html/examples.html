DiscreteWavelets Toolbox    

Examples

Each function documented in the DiscreteWavelets Toolbox comes with at least one example. You can either view (or hear!) the output, or you can cut and paste the included code into the Matlab workspace and test them yourself. Below are several examples that make use of many of the Toolbox functions.

Compute and Plot a One-Dimensional Wavelet Transformation

The DiscreteWavelets toolbox allows the user to perform many types of wavelet transformations. Here are a few.

A Haar wavelet transformation
An orthogonal wavelet transformation

Import and Plot a Digital Image

The DiscreteWavelets Toolbox comes equipped with 18 grayscale images and 9 color images. You can import and read these images, images of your own, or images you find on the internet.

Signal Denoising

There are many denoising functions contained in the DiscreteWavelets Toolbox. Following Algorithm 9.1 in the book, there are functions to perform every step of the algorithm. Alternatively, there is a routine that combines all tools to perform wavelet-based denoising via Algorithm 9.1.
