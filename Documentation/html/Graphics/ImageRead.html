DiscreteWavelets Toolbox

ImageRead

Import a digital image

Syntax

A = ImageRead(f)
A = ImageRead(f,options)

Description

A = ImageRead(f) reads the contents of a file f into A. If the image is grayscale, then A is a matrix. If the image is color, then A is a 3-dimensional array with A(:,:,1), A(:,:,2), A(:,:,3) holding the red, green, and blue portions of the image, respectively.

A = ImageRead(f,options) allows the user to set two parameters. See Optional Arguments for more details.

Optional Arguments

There are two optional arguments that can be set for ImageRead. The order of the parameters does not matter in the function call, but you must follow the standard convention 'Option', 'Value' when invoking the parameters.

PrintInfo

The option 'PrintInfo' can be set to 'True' (default is 'False') to instruct the function to print information about the image. In particular, if the option is set, the function prints the dimensions of the image, whether the image was read as grayscale or color, and the maximum number of iterations of a wavelet transformation that can be performed on the image.

PowersOfTwo

The 'PowersOfTwo' can be set to any nonnegative integer k. If set, the function removes the minimum number of rows from the bottom and columns from the right so that the dimensions of the resulting matrix is divisible by 2^k.

Remarks

The input f can be a file on disk or it can be a URL of an image.

Examples

Example 1

Read and plot an image. 
%Load all grayscale image files that come with the toolbox in cell array gry
gry=ImageNames('ImageType','GrayScale');
file = gry{7}; %Shows the name of the data file
A=ImageRead(file); %Load the image into matrix A
ImagePlot(A); %Plot the image

Example 2

Read an image and resize it so that five iterations of a wavelet transform can be performed on the resulting image. The commands 
%Load all color image files that come with the toolbox in cell array clr
clr=ImageNames('ImageType','Color','ListThumbnails','True');
file = clr{1} %Shows the name of the data file
%Load the image into matrix A and resize it so that we can perform 5 iterations
A=ImageRead(file,'PrintInfo','True','PowersOfTwo',5);
ImagePlot(A); %Plot the image
produces 
Chopping 16 columns off the right and 0 rows off the bottom of the image.

The dimensions of the image are 160 x 224 and the file was processed as a color image.

A total of 5 iterations of a wavelet transform can be performed on the image.

Example 3

Read an image from the internet and plot. The commands 
%Get an image from the internet
A=ImageRead('http://cam.mathlab.stthomas.edu/wavelets/CoverLarge.jpg','PrintInfo','True','PowersOfTwo',2);
ImagePlot(A); %Plot the image
produces 
Chopping 0 columns off the right and 1 rows off the bottom of the image.

The dimensions of the image are 452 x 300 and the file was processed as a color image.

A total of 2 iterations of a wavelet transform can be performed on the image.
