DiscreteWavelets Toolbox

ImageList

List of included image files

Syntax

DataList()

Description

ImageList() returns information about all image files that are included with the DiscreteWavelets toolbox.
ImageList(options) lets you specify whether you wish to see information on grayscale images or color images. The default is to give information on all images.

Remarks

You can add your own image files to the DiscreteWavelets Toolbox. Click here for more information.

The function returns the file name, the dimensions of the image, and the maximum number of iterations of wavelet transformations that can be performed for each data file that comes with the DiscreteWavelets Toolbox. The numbers in parentheses are values for the thumbnail versions of the images.

The command ImageList('ImageType','GrayScale') gives information about grayscale images.

The command ImageList('ImageType','Color') gives information about color images.

Example
