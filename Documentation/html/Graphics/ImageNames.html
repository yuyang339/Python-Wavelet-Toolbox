DiscreteWavelets Toolbox

ImageNames

Image file names

Syntax

[g,c] = ImageNames()
g = ImageNames('ImageType','Grayscale')
c = ImageNames('ImageType','Color')

Description

[g,c] = ImageNames() returns the absolute paths of each image file included in the DiscreteWavelets Toolbox.

[g.c] = ImageRead(f,options) allows the user to set two parameters. See Optional Arguments for more details.

Optional Arguments

There are two optional arguments that can be set for ImageNames. The order of the parameters does not matter in the function call, but you must follow the standard convention 'Option', 'Value' when invoking the parameters.

ImageType

The default output for ImageNames is two cell arrays. The first holds absolute path names for all grayscale images included in the toolbox while the second contains absolute pathnames for all color images included in the toolbox. The 'ImageType' option can be set to restrict output to only 'GrayScale' or 'Color'.

The command g=ImageNames('ImageType','GrayScale') gives only paths to grayscale images.

The command c=ImageNames('ImageType','Color') gives only paths to color images.

ListThumbnails

All images included with the DiscreteWavelets Toolbox come in two sizes. ImageNames, by default, gives path names to the large images in the set. If you want to obtain pathnames for the smaller (thumbnail) images, set the option 'ListThumbnails' to 'True' (default is 'False').

Remarks

You can use ImageNames to easily access image files included with the toolbox. It doesn't matter where you initially placed the DiscreteWavelets folder - as long as you did not rename it, ImageNames will find its location and return absolute path names to the image files.

You can add your own image files to the DiscreteWavelets Toolbox. Click here for more information.

Examples

Example 1

Get absolute paths and file names for all grayscale images included with the toolbox. 
%Load all grayscale image files that come with the toolbox in cell array gry
gry=ImageNames('ImageType','GrayScale');
file = gry{1} %Shows the name of the first image file

produces (on my computer)

C:\DiscreteWaveletsText\DiscreteWaveletsMatlab\Images\GrayScale\benches.png

Example 2

Get absolute paths and file names for all thumbnail color images included with the toolbox. 
%Load all color thumbnail image files that come with the toolbox in cell array clr
clr=ImageNames('ListThumbnails','True','ImageType','Color');
file = clr{1} %Shows the name of the first image file

produces (on my computer)

file =

C:\DiscreteWaveletsText\DiscreteWaveletsMatlab\Images\Color\bridge_small.png
