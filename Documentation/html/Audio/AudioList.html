DiscreteWavelets Toolbox

AudioList

List of included audio files

Syntax

AudioList()

Description

AudioList() returns information about all audio clips that are included with the DiscreteWavelets toolbox. There are no input arguments for this function.

Remarks

You can add your own audio files to the DiscreteWavelets Toolbox. Click here for more information.

The function returns the file name, audio type (.WAV or .AU), duration (in seconds), number of channels, sample rate, and number of samples for each audio clip that comes with the DiscreteWavelets Toolbox.
