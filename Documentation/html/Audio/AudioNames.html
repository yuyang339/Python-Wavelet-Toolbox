AudioNames

Audio file names

Syntax

a = AudioNames()

Description

a = AudioNames() returns the absolute paths of each audio file included in the DiscreteWavelets Toolbox.

Remarks

You can use AudioNames to easily access audio files included with the toolbox. It doesn't matter where you initially placed the DiscreteWavelets folder - as long as you did not rename it, AudioNames will find its location and return absolute path names to the audio files.

Example

Here's how you can use AudioNames to open and play a sound file that comes with the toolbox.

The following commands 

a=AudioNames(); %Load all audio files that come with the toolbox in cell array a
sndfile = a{2} %Shows the name of the audio file

produces (on my computer)

C:\Program Files\MATLAB\R2007b\toolbox\DiscreteWavelets\Sounds\bark.wav

Now use standard Matlab commands to load and play the file. 

[y, Fs] = wavread(sndfile); %Read the sound file
sound(y,Fs); %Play the sound file
