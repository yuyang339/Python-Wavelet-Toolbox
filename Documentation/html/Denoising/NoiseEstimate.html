NoiseEstimate

Estimate the noise level of a vector or matrix

Syntax

	s = NoiseEstimate(v,h)

Description

s=NoiseEstimate(v,h) takes as input a matrix or vector v and an orthogonal filter h and returns an estimate of the noise level of v.

Remarks

NoiseEstimate(v) is computed via by first computing one iteration of the wavelet transformation wt built from h. Then the level is estimated via the formula MAD(wt)/.6745. See Section 9.2 for more details.

Example

Create a noisy sine vector and estimate the noise level. 

x=0:.0001:.9999;
v=sin(2*pi*x); %Create a vector of sine values
n=randn(1,10000); %Create a noise vector
t=.15; %Set a noise level
y=v+t*n; %Create the noisy vector
s=NoiseEstimate(y,Coif(1))

The estimate is 

s =

    0.1503
