DiscreteWavelets Toolbox

MAD

Median absolute deviation

Syntax

	y = MAD(v)

Description

y=MAD(v) takes as input a matrix or vector v and returns the median absolute deviation of v.

Remarks

If v is a vector, MAD(v) is equivalent to the command y=median(abs(v-median(v))).

In the context of the DiscreteWavelets Toolbox, MAD is used to estimate the noise level of a noisy vector or matrix. See Section 9.2.

Example

Compute the median absolute deviation of a vector. 

v=round(10*rand(1,12)-5) %Create the vector
y=MAD(v) %Compute the median absolute deviation


v =

    -2     3    -2     3    -3     4    -2    -3    -2     1     0    -1


y =

    1.5000
