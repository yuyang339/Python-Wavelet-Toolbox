DiscreteWavelets Toolbox

UniversalThreshold

The universal threshold lambdauniv

Syntax

	lambda = UniversalThreshold(v,h,its)

Description

lambda = UniversalThreshold(v,h,its) takes a vector or matrix v, an orthogonal filter h, and a number of iterations its, and returns the universal threshold tolerance lambdauniv that is used in wavelet shrinkage denoising methods.

Remarks

The length or dimensions of v must be divisible by 2^its.

The filter h must be orthogonal and of even length.

The number of iterations its must be a positive integer.

The universal threshold is computing by first computing the wavelet transformation of v (using filter h) and then using that result to estimate the noise level sigma. The routine next determines the length n of a vector constructed by concatenating all highpass portions of the transform. The universal threshold is sigma*sqrt(2*log(n)).

For more information on the universal threshold, please see Section 9.2 of the book.

Example

Find the universal threshold for a vector. The commands 

x=0:.001:.999; %Create a list of evaluation points
v=sin(2*pi*x)+.15*randn(1,1000); %Construct a noisy vector
lambda=UniversalThreshold(v,Daub(4),3) %Compute the universal threshold

produce 

lambda =

    0.5900
