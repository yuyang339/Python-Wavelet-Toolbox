MaxIterations

Number of factors equal to 2 in the input

Syntax

	n = MaxIterations(v)

Description

n = MaxIterations(m) takes as input an integer m and returns integer n, where n is the largest integer such that 2^n divides m.

Remarks

The input m must be a positive integer.

MaxIterations(m) tells us the maximum number of iterations of a wavelet transformation that can be performed on a vector of length m.

We can use MaxIterations(m) tells us the maximum number of iterations of a wavelet transformation that can be performed on a r x c matrix. We compute n = min(MaxIterations(r),MaxIterations(c)).

Example

Find the maximum number of iterations of a wavelet transformation that can be performed on matrix A. The commands 
A=rand(80,96); %Create a 80x96 matrix
[r,c]=size(A);
n=min(MaxIterations(r),MaxIterations(c)) %Find the maximum number of iterations
B=HWT2D(A,n+1); %Try to do one more iteration and get an error
produce 
n =

     4

HWT2D: The number of iterations (5) is larger than the maximum iterations (4) allowed for 
the input.  Returning the input matrix.
