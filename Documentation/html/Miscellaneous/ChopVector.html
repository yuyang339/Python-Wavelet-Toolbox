ChopVector

Resize a vector

Syntax

	y = ChopVector(v,its)

	y = ChopVector(v,its,options)

Description

y=ChopVector(v,its) takes as input a vector v and a number of iterations its and chops a minimum number of elements off the end of the vector so that the length of y is divisible by 2^its.

y=ChopVector(v,its,'PrintInfo','True') performs the same action as the above statement but setting the 'PrintInfo' option to 'True' causes the function to print the length of y.

Example

Create a vector and make its length divisible by 8: 

x=rand(1,12); %Create a random vector
y=ChopVector(x,3)'; %Chop the vector

results in 

x =

    0.9575    0.9649    0.1576    0.9706    0.9572    0.4854    0.8003    0.1419    0.4218    0.9157    0.7922    0.9595


y =

    0.9575    0.9649    0.1576    0.9706    0.9572    0.4854    0.8003    0.1419
