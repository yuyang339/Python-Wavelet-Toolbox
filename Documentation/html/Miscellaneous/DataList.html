DiscreteWavelets Toolbox

DataList

List of included data files

Syntax

DataList()

Description

DataList() returns information about all data files that are included with the DiscreteWavelets toolbox. There are no input arguments for this function.

Remarks

You can add your own data files to the DiscreteWavelets Toolbox. Click here for more information.

The function returns the file name, the length of the file, and a description for each data file that comes with the DiscreteWavelets Toolbox.
