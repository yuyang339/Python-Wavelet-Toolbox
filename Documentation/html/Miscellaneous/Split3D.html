DiscreteWavelets Toolbox

Split3D

Split a 3-D array into three matrices

Syntax

	[R,G,B] = Split3D(A)

Description

[R,G,B] = Split3D(A) takes as input a three-dimensional array A and returns R=A(:,:,1), G=A(:,:,2), and B=A(:,:,3).

Remarks

[R,G,B]=Split3D(A) is used in the DiscreteWavelets Toolbox to take a 3-D array that is typically what is used to display color images in Matlab and split it into three color channels for processing.

Example

Read a color image, split it using Split3D, and plot the image and the three color channels. 
%Load all color images included with the toolbox
clr=ImageNames('ImageType','Color');
file = clr{4}; %Shows the name of the data file
A=ImageRead(file); %Load the image into 3-d array A
ImagePlot(A); %Plot the image
[R,G,B]=Split3D(A); %Split 3-D array into three matrices
figure
ImagePlot(R,'ChannelColor',[1 0 0]); %Plot the red channel
figure
ImagePlot(G,'ChannelColor',[0 1 0]); %Plot the green channel
figure
ImagePlot(B,'ChannelColor',[0 0 1]); %Plot the blue channel
