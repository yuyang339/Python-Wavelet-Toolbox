DiscreteWavelets Toolbox    

Introduction

The DiscreteWavelets Toolbox is a collection of functions written in Matlab to supplement the text Discrete Wavelet Transformations: An Elementary Approach with Applications (John J Wiley & Sons, Copyright © 2008) by Patrick J. Van Fleet.

The package contains modules for deriving orthogonal and biorthogonal wavelet filters and using them to compute one- and two-dimensional (biorthogonal) discrete wavelet transformations. Modules are also included so that the user can visualize digital images and their transformations as well as hear digital audio files and their transformations. Other modules allow the user to perform naive image compression or de-noise signals or audio clips. These routines include cumulative energy, entropy, PSNR, DCTs, gamma correction, histogram equalization, universal shrinkage tolerance, SureShrink tolerance, wavelet shrinkage, and SureShrink de-noising. There are also modules that allow the user to investigate Fourier series and perform color space conversions.

Help is available for all functions and optional arguments once the package has been installed. Each function help page also includes at least one example of how the function is used. Several of these examples are collected and available by clicking the Examples link. You can also learn about applications that use discrete wavelet transformations by accessing the Demos page.
