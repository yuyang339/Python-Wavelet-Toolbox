CDF97

Cohen, Daubechies, Feauveau biorthogonal filter pair

Syntax

	[h,ht] = CDF97()

Description

CDF97() returns the Cohen, Daubechies, Feauveau biorthogonal filter pair.

Example

The command 

[h,ht]=CDF97()

results in 

h = 
    0.0378
   -0.0238
   -0.1106
    0.3774
    0.8527
    0.3774
   -0.1106
   -0.0238
    0.0378

ht = 
   -0.0645
   -0.0407
    0.4181
    0.7885
    0.4181
   -0.0407
   -0.0645
