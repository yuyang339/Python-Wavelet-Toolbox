Daub

Daubechies orthogonal filter

Syntax

	h = Daub(n)

Description

Daub(n) returns the length n orthogonal Daubechies filter.

Remarks

n must be an even integer from the set {2,4,6,8,10,12,14}.

Daub calls an auxiliary function Daubfun in conjunction with Matlab's fsolve to find the desired orthogonal filter. The numerical solver is unstable for n>14.

Daub(2) is equivalent to Haar().

Example

The command 

h=Daub(6)

results in 

h = 
	 0.3327
	 0.8069
	 0.4599
	-0.350
	-0.0854
	 0.0352
