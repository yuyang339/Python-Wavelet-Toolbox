Coif

Coiflet orthogonal filter

Syntax

	h = Coif(n)

Description

Coif(n) returns the length 6n orthogonal Coiflet filter.

Remarks

n must be an even integer from the set {1,2,3}.

Daub calls an auxiliary function Coiffun in conjunction with Matlab's fsolve to find the desired orthogonal filter. The numerical solver is unstable for n>3.

Example

The command 

h=Coif(2)

results in 

h = 
   -0.0164
    0.0415
    0.0674
   -0.3861
   -0.8127
   -0.4170
    0.0765
    0.0594
   -0.0237
   -0.0056
    0.0018
    0.0007
