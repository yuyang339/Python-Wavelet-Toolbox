SplineFilters

Biorthogonal spline filter pair

Syntax

	[h,ht] = SplineFilters(n,nt)

Description

SplineFilters(n,nt) returns the (n,nt) biorthogonal spline filter pair.

Remarks

n and nt must be either both even or both odd.

The filters are symmetric. If n and nt are odd, then the filters are symmetric about h0 and ht0. If n and nt are even, then the filters satisfy hk = h1-k and htk = ht1-k.

Example

The command 

[h,ht]=SplineFilters(2,2)

results in the (5,3) biorthogonal spline filter pair 

h = 
   -0.1768
    0.3536
    1.0607
    0.3536
   -0.1768

ht = 
    0.3536
    0.7071
    0.3536

The command 

[h,ht]=SplineFilters(3,3)

results in the (8,4) biorthogonal spline filter pair 

h = 
    0.0663
   -0.1989
   -0.1547
    0.9944
    0.9944
   -0.1547
   -0.1989
    0.0663
ht = 
    0.1768
    0.5303
    0.5303
    0.1768
