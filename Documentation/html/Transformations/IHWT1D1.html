IHWT1D1

One-dimensional Haar inverse wavelet transformation, one iteration

Syntax

	v = IHWT1D1(wt)

Description

v = IHWT1D1(wt) returns one iteration of the one-dimensional Haar inverse wavelet transformation of wt.

Remarks

The length of wt must be an even integer.

Example

Here we compute one iteration of the one-dimensional Haar wavelet transformation. 

v=1:32; %Create a vector
wt=HWT1D1(v) %Compute the transformation
y=IHWT1D1(wt) %Compute the inverse
abs(y'-v) %Compare y to the original vector v
