DiscreteWavelets Toolbox

ILWT1D1

One-dimensional biorthogonal LeGall inverse wavelet transformation, one iteration

Syntax

	wt = ILWT1D1(v)

	wt = ILWT1D1(v,options)

Description

wt = LWT1D1(v) returns one iteration of the one-dimensional biorthogonal LeGall inverse wavelet transformation of wt.

wt = LWT1D1(v,'IntegerMap','True') returns one iteration of the one-dimensional biorthogonal LeGall inverse wavelet transformation of wt. Here it is assumed that the components of wt are integer-valued and as a result, the values of v are integer-valued as well (see Section 12.3 of the text).

Remarks

The length of wt must be an even integer.

Example

Here we compute one iteration of the one-dimensional biorthogonal LeGall inverse wavelet transformation. 

v=1:32; %Create a vector
wt=LWT1D1(v) %Compute the transformation
y=ILWT1D1(wt) %Compute the inverse
abs(y'-v) %Compare y to the original vector v

Here is the same example with the 'IntegerMap' option set to 'True'. 

v=1:32; %Create a vector
wt=LWT1D1(v,'IntegerMap','True') %Compute the transformation
y=ILWT1D1(wt,'IntegerMap','True') %Compute the inverse
abs(y'-v) %Compare y to the original vector v
