WT1D1

One-dimensional orthogonal wavelet transformation, one iteration

Syntax

	wt = WT1D1(v,h)

Description

wt = WT1D1(v,h) returns one iteration of the one-dimensional orthogonal wavelet transformation of v using orthogonal filter h.

Remarks

The length of v must be an even integer. The length of filter h must be an even integer.

Example

Here we compute one iteration of the one-dimensional orthogonal wavelet transformation. 

v=1:32; %Create a vector
h=Daub(4); %Load an orthogonal filter
wt=WT1D1(v,h) %Compute the transformation
