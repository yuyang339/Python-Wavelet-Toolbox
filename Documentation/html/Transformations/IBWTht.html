DiscreteWavelets Toolbox

IBWTht

Auxilary function used to compute one iteration of the biorthogonal inverse wavelet transformation

Syntax

y = IBWTht(u,f,passflag)

Description

y = IBWTht(u,f,passflag) is used by IBWT1D1 to compute the biorthogonal inverse wavelet transformation.

The biorthogonal inverse wavelet transformation can be written as W^T y. If we partition W into lowpass and highpass portions, and then partition y into two vectors u and w each of length(y)/2, we have 

	W^T y = [ H^T | G^T ] y = H^T u + G^T w

Depending on the filter, the function IBWTht computes either H^T u or G^T w. The third argument passflag indicates whether the computation involves the highpass portion or the lowpass portion.

Example

Here we compute one iteration of the one-dimensional biorthogonal inverse wavelet transformation of v using IBWTht. This is essentially the algorithm used by IBWT1D1 

v=1:32; %Create a vector
[h,ht]=SplineFilters(2,2); %Load the (5,3) spline filter pair
wt=BWT1D1(v,h,ht) %Compute the transformation
g=[1; -1; 1].*ht; %Build the highpass filter
u=wt(1:length(wt)/2); w=wt(length(wt)/2+1:end) %Split the vector wt
y=IBWTht(u',h,0)+IBWTht(w',g,1); %Compute the inverse transformation
abs(v-y)' %Compare with original vector v
