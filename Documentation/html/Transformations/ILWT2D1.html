DiscreteWavelets Toolbox

ILWT2D1

Two-dimensional biorthogonal LeGall inverse wavelet transformation, one iteration

Syntax

	A = ILWT2D1(wt)

	A = ILWT2D1(wt,options)

Description

A = ILWT2D1(wt) returns one iteration of the two-dimensional biorthogonal LeGall inverse wavelet transformation of wt.

A = ILWT2D1(wt,'IntegerMap','True') returns one iteration of the two-dimensional biorthogonal LeGall inverse wavelet transformation of wt. Here it is assumed that the components of wt are integer-valued and as a result, the values of A are integer-valued as well (see Section 12.3 of the text).

Remarks

The dimensions of wt must be even integers.

Example

Here we compute one iteration of the two-dimensional biorthogonal LeGall inverse wavelet transformation. 

A=rand(12,12); %Create a matrix
wt=LWT2D1(A) %Compute the transformation
B=ILWT2D1(wt) %Compute the inverse
abs(B-A) %Compare B to the original matrix A

Here is the same example with the 'IntegerMap' option set to 'True'. 

A=round(100*rand(12,12)); %Create a matrix
wt=LWT2D1(A,'IntegerMap','True') %Compute the transformation
B=ILWT2D1(wt,'IntegerMap','True') %Compute the inverse
abs(B-A) %Compare B to the original matrix A
