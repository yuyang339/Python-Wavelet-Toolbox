LeftHWT

The computation WA, where W is a Haar wavelet matrix

Syntax

	B = LeftHWT(A)

Description

B = LeftHWT(A) returns the produce WA where W is the Haar wavelet matrix.

Remarks

The dimensions of A must be even integers.

Example

Here we compute WA for a grayscale image. 

gry=ImageNames('ImageType','GrayScale'); %Get file names of grayscale images included with the package
A=ImageRead(gry{1}); %Read one of the images.
ImagePlot(A) %Plot the image

The result is

Now apply LeftHWT and plot the result. 

B=LeftHWT(A); %Compute WA
ImagePlot(B,'LinearScaling','LeftWT') %Plot the image
