IWTht

Auxilary function used to compute one iteration of the orthogonal inverse wavelet transformation

Syntax

y = IWTht(u,f)

Description

y = IWTht(u,f) is used by IWT1D1 to compute the orthogonal inverse wavelet transformation.

The orthogonal inverse wavelet transformation can be written as W^T y. If we partition W into lowpass and highpass portions, and then partition y into two vectors u and w each of length(y)/2, we have 

	W^T y = [ H^T | G^T ] y = H^T u + G^T w

Depending on the filter, the function IWTht computes either H^T u or G^T w.

Example

Here we compute one iteration of the one-dimensional orthogonal inverse wavelet transformation of v using IWTht. This is essentially the algorithm used by IBWT1D1 

v=1:8; %Create a vector
h=Daub(4); %Load the Daubechies 4-term orthogonal filter
hh=flipud(h); %Reverse the filter
wt=WT1D1(v,h) %Compute the transformation
g=[-1; 1; -1; 1].*h; %Build the highpass filter
u=wt(1:length(wt)/2); w=wt(length(wt)/2+1:end) %Split the vector wt
y=IWTht(u',hh)+IWTht(w',g); %Compute the inverse transformation
abs(v-y') %Compare with original vector v
