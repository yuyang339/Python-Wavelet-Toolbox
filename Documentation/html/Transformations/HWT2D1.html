HWT2D1

Two-dimensional Haar wavelet transformation, one iteration

Syntax

	wt = HWT2D1(A)

Description

wt = HWT2D1(A) returns one iteration of the two-dimensional Haar wavelet transformation of A.

Remarks

The dimensions of A must be even integers.

Example

Here we compute one iteration of the two-dimensional Haar wavelet transformation. 

A=rand(12,12); %Create a matrix
wt=HWT2D1(A) %Compute the transformation
