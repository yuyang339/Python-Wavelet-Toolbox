RightHWT

The computation AW^T, where W is a Haar wavelet matrix

Syntax

	B = RightHWT(A)

Description

B = RightHWT(A) returns the produce AW^T where W is the Haar wavelet matrix.

Remarks

The dimensions of A must be even integers.

Example

Here we compute AW^T for a grayscale image. 

gry=ImageNames('ImageType','GrayScale'); %Get file names of grayscale images included with the package
A=ImageRead(gry{1}); %Read one of the images.
ImagePlot(A) %Plot the image

The result is

Now apply RightHWT and plot the result. 

B=RightHWT(A); %Compute AW^T
ImagePlot(B,'LinearScaling','RightWT') %Plot the image
