LWT1D

One-dimensional biorthogonal LeGall wavelet transformation, multiple iterations

Syntax

	wt = LWT1D(v,its)

	wt = LWT1D(v,its,options)

Description

wt = LWT1D(v,its) returns its iterations of the one-dimensional biorthogonal LeGall wavelet transformation of v.

wt = LWT1D(v,its,'IntegerMap','True') returns its iterations of the one-dimensional biorthogonal LeGall wavelet transformation of v. Here it is assumed that the components of v are integer-valued and as a result, the values of wt are integer-valued as well (see Section 12.3 of the text).

Remarks

The length of v must be divisible by 2^its.

Example

Here we compute three iterations of the one-dimensional biorthogonal LeGall wavelet transformation. 

v=1:32; %Create a vector
wt=LWT1D(v,3) %Compute the transformation

Here is the same example with the 'IntegerMap' option set to 'True'. 

v=1:32; %Create a vector
wt=LWT1D(v,3,'IntegerMap','True') %Compute the transformation
