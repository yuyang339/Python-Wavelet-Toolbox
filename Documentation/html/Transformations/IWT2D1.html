IWT2D1

Two-dimensional orthogonal inverse wavelet transformation, one iteration

Syntax

	A = IWT2D1(wt,h)

Description

A = IWT2D1(wt,h) returns one iteration of the two-dimensional orthogonal inverse wavelet transformation of wt using orthogonal filter h.

Remarks

The dimensions of wt must be even integers. The length of filter h must be an even integer.

Example

Here we compute one iteration of the two-dimensional orthogonal inverse wavelet transformation. 

A=rand(12,12); %Create a matrix
h=Daub(6); %Load an orthogonal filter
wt=WT2D1(A,h) %Compute the transformation
B=IWT2D1(wt,h) %Compute the transformation
abs(B-A) %Compare B to the original matrix A
