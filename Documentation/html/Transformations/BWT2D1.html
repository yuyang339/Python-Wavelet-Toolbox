DiscreteWavelets Toolbox

BWT2D1

Two-dimensional biorthogonal wavelet transformation, one iteration

Syntax

	wt = BWT2D1(A,h,ht)

	wt = BWT1D1(A,h,ht,options)

Description

wt = BWT2D1(A,h,ht) returns one iteration of the two-dimensional biorthogonal wavelet transformation of A using symmetric biorthogonal filters h and ht.

wt = BWT2D1(A,h,ht,'Boundary','Reflective') returns one iteration of the two-dimensional biorthogonal wavelet transformation of A using symmetric biorthogonal filters h and ht, but exploits the fact that the filters are symmetric to produce a transformation that better handles edge effects. See Section 11.3 in the text for more details.

Remarks

The dimensions of A must be even integers.

You must supply the routine with a symmetric pair of biorthogonal filters. That is, in the case when the lengths of both h and ht are odd, then the filters must be symmetric about h0 and ht0. In the case when the lengths are even, then the filters should satisfy hk = h1-k and htk = ht1-k.

Example

Here we compute one iteration of the two-dimensional biorthogonal wavelet transformation of matrix A. 

A=rand(10,10) %Create a matrix - must be of even dimensions.
[h,ht]=SplineFilters(2,2); %Load the (5,3) spline filter pair
wt=BWT2D1(A,h,ht) %Compute the transformation

Here is the same example but utilizes the boundary conditions (see Section 11.3 of the text). 

A=rand(10,10) %Create a matrix - must be of even dimensions.
[h,ht]=SplineFilters(2,2); %Load the (5,3) spline filter pair
wt=BWT2D1(A,h,ht,'Boundary','Reflective') %Compute the transformation
