RightWT

The computation AW^T, where W is an orthogonal wavelet matrix

Syntax

	B = RightWT(A,h)

Description

B = RightWT(A,h) returns the produce AW^T where W is an orthogonal wavelet matrix constructed from filter h.

Remarks

The dimensions of A must be even integers. The orthogonal filter h must be of even length.

Example

Here we compute AW^T for a grayscale image. 

gry=ImageNames('ImageType','GrayScale'); %Get file names of grayscale images included with the package
A=ImageRead(gry{11}); %Read one of the images.
ImagePlot(A) %Plot the image

The result is

Now apply RightWT and plot the result. 

B=RightHWT(A,Daub(4)); %Compute WA
ImagePlot(B,'LinearScaling','RightWT') %Plot the image
