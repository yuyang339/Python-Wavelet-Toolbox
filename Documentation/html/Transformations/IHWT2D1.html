IHWT2D1

Two-dimensional Haar inverse wavelet transformation, one iteration

Syntax

	A = IHWT2D1(wt)

Description

A = IHWT2D1(wt) returns one iteration of the two-dimensional Haar inverse wavelet transformation of wt.

Remarks

The dimensions of wt must be even integers.

Example

Here we compute one iteration of the two-dimensional Haar wavelet transformation. 

A=rand(12,12); %Create a matrix
wt=HWT2D1(A) %Compute the transformation
B=IHWT2D1(wt) %Compute the inverse
abs(B-A) %Compare B to the original matrix A
