WT2D

Two-dimensional orthogonal wavelet transformation, multiple iterations

Syntax

	wt = WT2D(A,h,its)

Description

wt = WT2D(A,h,its) returns its iterations of the two-dimensional orthogonal wavelet transformation of A using orthogonal filter h.

Remarks

The dimensions of A must be integers divisible by 2^its. The length of filter h must be an even integer.

Example

Here we compute two iterations of the two-dimensional orthogonal wavelet transformation. 

A=rand(12,12); %Create a matrix
h=Coif(1); %Load an orthogonal filter
wt=WT2D(A,h,2) %Compute the transformation
