IWT1D1

One-dimensional orthogonal inverse wavelet transformation, one iteration

Syntax

	v = IWT1D1(wt,h)

Description

v = IWT1D1(wt,h) returns one iteration of the one-dimensional orthogonal inverse wavelet transformation of wt using orthogonal filter h.

Remarks

The length of wt must be an even integer. The length of filter h must be an even integer.

Example

Here we compute one iteration of the one-dimensional orthogonal inverse wavelet transformation. 

v=1:32; %Create a vector
h=Daub(4); %Load an orthogonal filter
wt=WT1D1(v,h) %Compute the transformation
y=IWT1D1(wt,h) %Compute the transformation
abs(y'-v) %Compare y to the original vector v
