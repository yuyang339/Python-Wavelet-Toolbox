DiscreteWavelets Toolbox

BWT1D

One-dimensional biorthogonal wavelet transformation, multiple iterations

Syntax

	wt = BWT1D(v,h,ht,its)

	wt = BWT1D1(v,h,ht,its,options)

Description

wt = BWT1D(v,h,ht,its) returns its iterations of the one-dimensional biorthogonal wavelet transformation of v using symmetric biorthogonal filters h and ht.

wt = BWT1D(v,h,ht,'Boundary','Reflective') returns its iterations of the one-dimensional biorthogonal wavelet transformation of v using symmetric biorthogonal filters h and ht, but exploits the fact that the filters are symmetric to produce a transformation that better handles edge effects. See Section 11.3 in the text for more details.

Remarks

The length of v must be divisible by 2^its.

You must supply the routine with a symmetric pair of biorthogonal filters. That is, in the case when the lengths of both h and ht are odd, then the filters must be symmetric about h0 and ht0. In the case when the lengths are even, then the filters should satisfy hk = h1-k and htk = ht1-k.

Example

Here we compute 3 iterations of the one-dimensional biorthogonal wavelet transformation. 

v=1:32; %Create a vector make sure its length is divisible by 2^3.
[h,ht]=SplineFilters(2,2); %Load the (5,3) spline filter pair
wt=BWT1D(v,h,ht,3) %Compute the transformation

Here is the same example using the optional argument to handle boundary conditions (see Section 11.3 of the text). 

v=1:32; %Create a vector make sure its length is divisible by 2^3.
[h,ht]=SplineFilters(2,2); %Load the (5,3) spline filter pair
wt=BWT1D(v,h,ht,3,'Boundary','Reflective') %Compute the transformation
