LWT1D1

One-dimensional biorthogonal LeGall wavelet transformation, one iteration

Syntax

	wt = LWT1D1(v)

	wt = LWT1D1(v,options)

Description

wt = LWT1D1(v) returns one iteration of the one-dimensional biorthogonal LeGall wavelet transformation of v.

wt = LWT1D1(v,'IntegerMap','True') returns one iteration of the one-dimensional biorthogonal LeGall wavelet transformation of v. Here it is assumed that the components of v are integer-valued and as a result, the values of wt are integer-valued as well (see Section 12.3 of the text).

Remarks

The length of v must be an even integer.

Example

Here we compute one iteration of the one-dimensional biorthogonal LeGall wavelet transformation. 

v=1:32; %Create a vector
wt=LWT1D1(v) %Compute the transformation

Here is the same example with the 'IntegerMap' option set to 'True'. 

v=1:32; %Create a vector
wt=LWT1D1(v,'IntegerMap','True') %Compute the transformation
