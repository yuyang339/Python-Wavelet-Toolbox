LWT2D

Two-dimensional biorthogonal LeGall wavelet transformation, multiple iterations

Syntax

	wt = LWT2D(A,its)

	wt = LWT2D1(A,its,options)

Description

wt = LWT2D1(A,its) returns its iterations of the two-dimensional biorthogonal LeGall wavelet transformation of A.

wt = LWT2D1(A,its,'IntegerMap','True') returns its iterations of the two-dimensional biorthogonal LeGall wavelet transformation of A. Here it is assumed that the components of A are integer-valued and as a result, the values of wt are integer-valued as well (see Section 12.3 of the text).

Remarks

The dimensions of A must be divisible by 2^its.

Example

Here we compute two iteration of the two-dimensional biorthogonal LeGall wavelet transformation. 

A=rand(12,12); %Create a matrix
wt=LWT2D(A,2) %Compute the transformation

Here is the same example with the 'IntegerMap' option set to 'True'. 

A=rand(12,12); %Create a matrix
wt=LWT2D(A,2,'IntegerMap','True') %Compute the transformation
