HWT1D

One-dimensional Haar wavelet transformation, multiple iterations

Syntax

	wt = HWT1D(v,its)

Description

wt = HWT1D(v,its) returns its iterations of the one-dimensional Haar wavelet transformation of v.

Remarks

The length of v must be divisible by 2^its.

Example

Here we compute three iterations of the one-dimensional Haar wavelet transformation. 

v=1:32; %Create a vector
wt=HWT1D(v,3) %Compute the transformation
